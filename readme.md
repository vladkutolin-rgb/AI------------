AI-Generator: Автоматическая генерация презентаций с помощью нейросетей
1. Название проекта
AI-Generator — сервис автоматической генерации презентаций на основе текстового запроса и загруженного документа с использованием нейросетевых моделей через API.

2. Проблема
Создание качественных презентаций — трудоёмкий процесс, который занимает у сотрудников компаний от 2 до 6 часов на подготовку 10–15 слайдов. Требуется анализ исходных материалов, структурирование информации, написание текстов, подбор визуального оформления и вёрстка. Большинство сотрудников не владеют дизайнерскими навыками на достаточном уровне, а существующие западные решения (Gamma, Beautiful.ai) недоступны в России из-за санкций, не принимают российские карты и плохо работают с кириллицей.

3. Решение
Мы разработали веб-сервис на Python (FastAPI), который позволяет пользователю:

Ввести текстовый запрос с описанием желаемой презентации

Загрузить документ (PDF или DOCX) как источник контента

Получить готовую презентацию в формате PPTX с текстом, дизайном и сгенерированными нейросетью изображениями

Сервис использует три нейросетевые модели через API Ростелекома (ai.rt.ru):

LLM (Llama Qwen 2.5 72B) — анализирует входные данные и формирует структуру слайдов: заголовки, текстовые блоки, ключевые слова для изображений

YandexArt / Stable Diffusion — генерирует иллюстрации для каждого слайда по ключевым словам

Программная сборка PPTX через библиотеку python-pptx с адаптивным дизайном

4. Ключевые функции (MVP)
🌐 Веб-интерфейс в фирменном стиле Ростелекома

📝 Одно поле ввода — пользователь описывает презентацию своими словами

📄 Загрузка PDF/DOCX — автоматическое извлечение текста

🧠 AI-анализ — нейросеть сама определяет количество слайдов, тему оформления, акцентные цвета

🎨 Адаптивный дизайн — светлая/тёмная тема, автоматический подбор размера шрифта под объём текста

🖼️ Генерация изображений — каждый слайд получает уникальную картинку по теме

📊 Экспорт в PPTX — готовый файл PowerPoint с сохранением структуры
